# Redeemed in Chains Bible Study Companion

Static Bible study and reference app for `Redeemed in Chains`.

## Interpretation Preference

This app is intentionally shaped for a conservative evangelical, literal interpretation of Scripture:

- Scripture is the final authority.
- Biblical words like sin, guilt, repentance, grace, forgiveness, and reconciliation are treated as theological realities, not merely therapeutic metaphors.
- Application follows interpretation instead of replacing it.
- Grace is never used to minimize consequence, victims, or moral responsibility.

## Current Source Status

The supplied final PDF manuscript was OCR-reviewed locally and used to build the chapter map and study modules. The full manuscript text is not embedded in the app or deploy zip.

## Local Preview

Open `index.html` directly in a browser, or serve this folder with a static server.
